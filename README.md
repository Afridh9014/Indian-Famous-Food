This Power BI dashboard visualizes India’s diverse and famous cuisines across different states and regions. The goal of this project is to provide insights into Indian food culture by showcasing regional specialties, ingredient trends, and the distribution of vegetarian vs non-vegetarian dishes across the country.

🗂️ Dataset

Dataset Name: indian_food.csv Source: Open-source dataset containing details of Indian dishes. Key Columns:

name – Name of the dish

state – State/region where the dish is popular

diet – Type of diet (Vegetarian/Non-Vegetarian)

prep_time – Preparation time (in minutes)

cook_time – Cooking time (in minutes)

flavor_profile – Taste category (spicy, sweet, sour, etc.)

course – Course type (main course, dessert, snack, etc.)

ingredients – Key ingredients used

🎯 Objectives

Visualize regional distribution of famous Indian foods.

Compare vegetarian vs non-vegetarian food trends across states.

Analyze average cooking and preparation times by cuisine type.

Identify the most common flavor profiles and courses across regions.

Highlight popular dishes from each Indian state.

⚙️ Tools & Technologies

Power BI – For data visualization and dashboard creation.

Microsoft Excel / CSV – For data preprocessing and cleaning.

GitHub – For project documentation and sharing.

📈 Key Insights (from Dashboard)

North Indian cuisine shows a higher proportion of spicy and non-vegetarian dishes.

South Indian states have more vegetarian and rice-based dishes.

Desserts dominate the West and East regions (e.g., Maharashtra, West Bengal).

The average preparation time varies widely by course and flavor type.
